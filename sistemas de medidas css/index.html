¿Qué son las unidades de medida en CSS?

Las unidades de medida en CSS permiten definir tamaños, márgenes, espacios y otros aspectos visuales en una página web , existen unidades absolutas y relativas que se usan según el contexto.

Unidades Absolutas

Las unidades absolutas tienen un valor fijo que no depende del entorno, ejemplos incluyen: píxeles (px), centímetros (cm), milímetros (mm), pulgadas (in), puntos (pt) y picas (pc).

Unidades Relativas

Las unidades relativas cambian dependiendo del contexto. Ejemplos son: porcentaje (%), em, rem, vw (ancho del viewport) y vh (alto del viewport). Son clave para diseños adaptables o "responsive".
